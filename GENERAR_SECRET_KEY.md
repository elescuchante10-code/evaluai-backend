# Generar SECRET_KEY

Como no tienes Python instalado en el terminal, aquí tienes una clave segura generada:

```
a8f7d9e3b2c1a0f5e4d8c7b6a5f4e3d2c1b0a9f8e7d6c5b4a3f2e1d0c9b8a7f6e5d4c3b2a1
```

## Instrucciones:

1. Abre el archivo `backend/.env`
2. Busca esta línea:
   ```
   SECRET_KEY=tu-clave-secreta-super-larga-para-jwt-minimo-32-caracteres
   ```

3. Reemplázala con:
   ```
   SECRET_KEY=a8f7d9e3b2c1a0f5e4d8c7b6a5f4e3d2c1b0a9f8e7d6c5b4a3f2e1d0c9b8a7f6e5d4c3b2a1
   ```

4. Guarda el archivo

¡Listo para deploy!
